# 🎵 Discord Music Bot

## Setup
1. Copy `.env.example` to `.env`
2. Add your Discord bot token
3. Run `npm install`
4. Run `npm run dev`

## Commands
- `!play <song>` - Play a song
- `!skip` - Skip current song
- `!pause` / `!resume` - Pause/Resume
- `!stop` - Stop and clear queue
- `!queue` - View queue
- `!list` - List all songs
- `!np` - Now playing
- `!join` - Join voice channel

## Add Songs
Just attach an MP3 file in Discord chat — bot will save it automatically!
